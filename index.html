The Road to Enlightenment

Littering a dark and dreary road lay the past relics of browser-specific tags, incompatible DOMs, broken CSS support, and abandoned browers.

We must clear the mind of the past. Web englightenment has been achieved thanks to tireless efforts of folk like thw WSC, WaSP, and the major brower creators.

the CSS Zen Garden invites you to relax and meditate on importatnt lessons of the masters. Begin to see with clarity. Learn to use the time-honored techniques in new invigorating fashion. become one with the web

So What Is This About

There is a continuing need to show the power of CSS. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view some of the existing designs in the list. Clicking on any one will load the style sheet into this very page. the HTML reamins the same, the only thing that has changed is the external CSS file. Yes, really.

CSS allows complete and total control over the style of hypertext document. The only way this can be illustrated in a way that gets the people excited is by demonstrating wehat can truly be, once the reins are placed in the hands of thise able to create beauty from strcuture. Designers and coders alike have contributed to the beauty of teh web; we can always push it further

Participation

Strong visual design has aleays been our focus. You are modifying this page, so strong CSS skills are necessary too, but the example files are commented well enough that even CSS novices can use them as starting points. Please see the CSS Resource Guide fro advance tutorials and tips on working with CSS

You may modify the style sheet in any way you wish, but not the HTML. This may seem daunting at first if you've never worked this way before, but follow the listed links to learn more, and use the sample files as a guide.

Download the sample HTML and CSS to work on a copy localy. Once you have completed your masterpiece (and please, dont submit half-finished work) upload your CSS file to web server under your control. Send us a link to archive of that file and all associated assets, and if we choose to use it we will downlaod it and aplce it on our server.

Benefits

Why participate? For recognition, inspiration, and a rescource we can all refer to showing people how ammazing CSS realy can be. This site serves as equal parts inspiration for those working on the web today, learning tool for those who will be tomorrow, and gallery of future techniques we can all look forard to.
